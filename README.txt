## Cài đặt ở local:

1. Ứng dụng chạy trên nền Nodejs, vui lòng cài Nodejs Version: >=16.1.10 trên local.

2. Cài đặt các pagekage cần thiết
Đứng ở thư mục root, "gootax-customer-driver", cài đặt `node_modules`, mở command line gõ:
npm install

3. Chạy ứng dụng
Sau khi các pagekage được cài đặt xong, để chạy chương trình gõ:
node index

4. Mở ứng dụng trên trình duyện qua link: http://localhost:2000
